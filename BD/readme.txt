Desacargue los archivos desde https://www.nexusmods.com/skyrimspecialedition/mods/4505?tab=files
las carpetas que inician con "0 - " son para los String del juego base, actualmente están para español y ruso creadas las carpetas pero si necesita una nueva solo mantenga el formato:
0 - [ idioma ] Original Strings

para los DLC se usara "1 - "
1 - [ idioma ] DLC Strings

Debe descomprimir los archivos dentro de estas carpetas asegurando que esten los direcciones finales de esta forma:
0 - [ idioma ] Original Strings\strings
1 - [ idioma ] DLC Strings\strings
